wtf22-day1-b Non-Overlapping Swaps

发表于 2024-03-07 分类于 Solutions 阅读次数：

Description

你有一个长度为 nnn 的排列 PPP。 你希望用不超过 n−1n-1n−1 次交换操作将其排序， 同时满足以下条件：设相邻两次交换操作交换的数所在位置下标分别为 (a,b)(a,b)(a,b)和 (c,d)(c,d)(c,d)（a≤b,c≤da\le b,c\le da≤b,c≤d)， 那么要求 b≤cb\le cb≤c 或者 d≤ad\le ad≤a。可以交换同一位置的数。现在要求你构造出一个合法解， 可以证明一定有解。多测， n≤250000n\le 250000n≤250000。

Solution

建图 i→pii\to p_ii→pi​ 表示 iii 位置上的数应该放到 pip_ipi​ 位置上。

多个环可以用 (1,1)(1, 1)(1,1) 拼接。

对于一个环，把环上所有位置拿出来，让最小的两个位置为 x,y(x<y)x, y(x < y)x,y(x<y)，尝试构造 (x,y)(x, y)(x,y) 一次操作。

把连向 xxx 的边断开，把橙色的环递归做。

然后交换 (x,y)(x, y)(x,y)。

把 xxx 连出去的边断开，蓝色的环递归做。

这样保证了任意时刻递归的操作都在 (x,y)(x, y)(x,y) 右边。

实现考虑钦定最小的位置为第一个，断环成链建笛卡尔树，直接递归即可。

时间复杂度 O(n)O(n)O(n)。
