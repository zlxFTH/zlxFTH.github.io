CF578F Mirror Box

发表于 2024-03-18 分类于 Solutions 阅读次数：

Description

在一个 n×mn \times mn×m 的网格中，每个格子里都有一个呈 \ 或 / 状的镜子。

一个合法的网格需要满足从任意一个边界段垂直射进网格中，光线会从相邻的边界段射出，同时网格中的每一段都被至少一条光线穿透。

现在网格中有 kkk 个位置的镜子形状不确定，求有多少种合法的网格。

n,m≤100n,m \le 100n,m≤100，k≤200k \le 200k≤200，答案对质数 ppp 取模。

Solution

还是太神仙了。

套路地进行网格图黑白染色，每条镜子可以看作边。

发现条件转换为：

  • 不存在环
  • 黑边或者白边构成一棵树

不存在环是因为有环只有一定有某一段不被穿透，以黑点为例（假设只考虑黑点），如果中间的黑点不和其他黑点有边，那么一定存在环，于是每个中间的黑点一定存在一条路径走到边界上的黑点，对于相邻两个边界段的两个黑色端点，他们如果不连通，一定可以把整个网格图 split 成两部分使得他们不连通，中间这部分一定有超过 222 个边界段并且一定不能使得相邻两段两两匹配（证明我不懂啊，画下图确实是），否则就可以考虑找到这两个点在这棵树上的路径，可以观察到一些规律，反射一定是从两端往中间进行的。

在黑边确定是一棵树的情况下，白边有且仅有一种方案。

于是并查集缩点之后转换成了 Matrix-Tree 定理模板。

时间复杂度 O(k3)O(k^3)O(k3)。
