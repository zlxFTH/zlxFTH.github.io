D. “a” String Problem

去掉末尾 aaa，所有合法串一定是最右边一个非 aaa 位置结尾的后缀，暴力枚举复杂度是调和级数，Hash 判断即可。

E. Shuffle

不考虑根的特殊贡献，相邻两个点最多贡献一个，于是上界为最大独立集，容易构造。

换根 DP 即可。

F. Reconstruction

长度为 lenlenlen 的区间值域限制为 [−len×m,len×m][-len\times m, len\times m][−len×m,len×m]。

值域限制包含可拆分，只用考虑相邻两个位置的限制：

  • 同为 P/SP/SP/S：限制长度为 111 的值域
  • P,SP, SP,S：限制了整个序列的和
  • S,PS, PS,P：限制了整个序列的和加上长度为 222 的值域

考虑枚举开头一段 PPPPPSPPPPPSPPPPPS 即可求出和，剩下直接 DP 判断。

G. Magic Trick II

直观理解答案离 nnn 不会太远，ans=n/n−1ans = n/n - 1ans=n/n−1​ 容易判断。

对于 ans=n−2ans = n - 2ans=n−2，可以实现整体向左循环移 222 位，后 n−1n - 1n−1 个数循环右移 111 位，可以镜像。

nnn 为奇数，可以实现整体向左循环移 111 位，于是可以考虑逐步让 1∼i1\sim i1∼i 连续，每次将 i+1i + 1i+1 循环移到开头，1∼i1\sim i1∼i 循环移到末尾。

nnn 为偶数执行上述或者镜像操作一定能使 1∼n1\sim n1∼n 连续，但不能保证 111 循环移动到开头，这里是因为操作不改变逆序对奇偶性。

该情况如果不行则 ans=n−3ans = n - 3ans=n−3，可以把 nnn 移动到末尾转换成 n−1n - 1n−1 的奇数情况。
