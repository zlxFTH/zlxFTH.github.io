ABC219H Candles

发表于 2024-02-17 分类于 Solutions 阅读次数：

Description

在一条数轴上有 NNN 支蜡烛，第 iii 支蜡烛长度为 AiA_iAi​，且位于数轴上的 XiX_iXi​ 处。

时刻 000 时，NNN 支蜡烛都被点燃。一根点燃的蜡烛每过一时刻就会减少 111 的长度。当长度变为 000 时蜡烛将会熄灭。熄灭的蜡烛的长度不会减少。

时刻 000 时，你位于数轴上的 000 处。一时刻中你可以在数轴上左右移动不超过 111 的单位距离。当你到达某支蜡烛所在的位置时，你可以选择熄灭这支蜡烛，熄灭蜡烛的时间忽略不计。如果有多支蜡烛位于同一个位置，你可以一次性熄灭该位置上的所有蜡烛。

求时刻 1010010^{100}10100 时最大的蜡烛总长度。

1 ≤ N ≤ 3001\ \leq\ N\ \leq\ 3001 ≤ N ≤ 300

Solution

据说叫关路灯 dp。

每个蜡烛的贡献是 max⁡(ai−t,0)\max(a_i - t, 0)max(ai​−t,0)。

设 f(l,r,0/1,t)f(l, r, 0/1, t)f(l,r,0/1,t) 表示暂停了 [l,r][l, r][l,r] 的蜡烛，当前在左/右，当前时间为 ttt 的答案。 ttt 这一维没有办法优化掉。

先把 ∑ai\sum a_i∑ai​ 加上，然后每走一步的代价就是当前没有熄灭的蜡烛，直接减掉。但是这样做可能会让某些蜡烛变成负数，但是负数是一定不优的，所以可以钦定那些蜡烛可以计算贡献。

钦定的过程加在 dp 转移中：设 f(l,r,k,0/1)f(l, r, k, 0/1)f(l,r,k,0/1) 表示暂停了 [l,r][l, r][l,r] 的蜡烛，区间外还有 kkk 个要计算贡献的蜡烛，当前在左/右的贡献。枚举拓展到 l−1/r+1l - 1/r + 1l−1/r+1 和该蜡烛是否计算贡献即可。

时间复杂度 O(n3)O(n^3)O(n3)。
