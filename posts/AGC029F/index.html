AGC029F Construction of a tree

发表于 2023-12-07 更新于 2024-03-07 分类于 Solutions 阅读次数：

Description

给定 n−1n-1n−1 个点集（全集为 {1,2,…,n}\{1,2,\ldots,n\}{1,2,…,n}），从每个集合内选两个点连边，使得最后形成一棵树。输出方案。

n−1n-1n−1 条边按顺序对应这 n−1n-1n−1 个集合输出。

n≤105n \leq 10^5n≤105，∑∣S∣≤2×105\sum |S| \leq 2 \times 10^5∑∣S∣≤2×105。

Solution

钦定以 111 为根，亲定每个集合一定是儿子连父亲，那么给每个点找父亲的过程首先就是要将 2∼n2\sim n2∼n 这些点和 n−1n - 1n−1 个集合进行完美匹配。

找出了一个完美匹配之后如何进行构造，考虑 bfs，从 111 开始，遍历所有包含 111 的集合，然后把这些集合匹配的点连向 111，加入队列依次构造。

如果不存在完美匹配或者不能 bfs 构造则一定无解。

bfs 是否有把有解判成无解的情况？其实是没有的，当无解一定存在一个点集，这个点集对应的集合的并依旧是这个点集，那一定不合法，否则一定能构造出来。

匹配上个 Dinic 最大流即可。
